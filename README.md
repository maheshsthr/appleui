# 🍎 AppleUI

A responsive Apple Education Store inspired landing page built using HTML and CSS.

## ✨ Features

- 📱 Responsive Design
- 🎨 Clean Apple-like UI
- 🖥️ Product Showcase Cards
- ↔️ Horizontal Card Scroll
- 📲 Mobile Optimized Layout
- 🚫 Hidden Scrollbars
- ⚡ Smooth Scrolling

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## 🚀 Run Project

```bash
git clone https://github.com/maheshsthr/appleui.git
```

Open `index.html` in browser.

## 📚 Learning Focus

- Flexbox
- Responsive Design
- Media Queries
- Scrollable Layouts
- UI Cloning
